1. How to Configure Database & Back-end on Your Server?

  1. First of all find the Radio folder from the downloaded package and upload to your server.
  2. Create a new database from cPanel
  3. Create new user for the database & give all the privileges to the user
  4. Import the database file 'radio.sql' to your PHPMYADMIN
  5. Open File Manager or FTP or File Explorer
  6. If you are using SSL (HTTPS) than go to 'application/config/' directory and Open 'config.php' file in PHP Code and Set http to https on line no - 27.
  7. Go To 'application/config/' directory and Open 'database.php' file in PHP Code
  8. Set database credentials with your credentials on line number - 51,52,53,54
    $db['default']['hostname'] = 'localhost'; // Change as required
    $db['default']['username'] = "mydb_user"; // Change as required
    $db['default']['password'] = "mydb_user_password"; // Change as required
    $db['default']['database'] = "myradiopanel_db"; // Change as required

Thats it! Now you are ready to run your first Radio App : http://radio.wrteam.in
